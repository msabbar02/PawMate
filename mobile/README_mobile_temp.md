# 📱 PawMate Mobile App

Aplicación móvil para Android e iOS desarrollada con React Native y Expo.

## 🚀 Tecnologías

- React Native
- Expo
- React Navigation
- Firebase (Authentication, Firestore, Storage)
- AsyncStorage

## 📂 Estructura (Próximamente)

```
mobile/
├── src/
│   ├── screens/       # Pantallas de la app
│   ├── components/    # Componentes reutilizables
│   ├── navigation/    # Configuración de navegación
│   ├── services/      # Servicios (Firebase, API)
│   ├── utils/         # Utilidades
│   └── constants/     # Constantes y configuración
├── assets/            # Imágenes, fuentes, iconos
└── App.js            # Punto de entrada
```

## 🎯 Funcionalidades

- Autenticación de usuarios
- Gestión de perfiles de mascotas
- Seguimiento de salud y vacunas
- Comunidad y posts
- Tienda de productos
- Notificaciones

## 🔧 Instalación

```bash
cd mobile
npm install
```

## ▶️ Ejecutar

```bash
# Desarrollo
npm start

# Android
npm run android

# iOS
npm run ios
```
