# 👨‍💼 PawMate Admin Panel

Panel de administración completo para gestionar la plataforma PawMate.

## 🚀 Tecnologías

- React / Next.js
- Firebase Admin SDK
- Material-UI / Ant Design
- Chart.js / Recharts (para gráficos)

## 📂 Estructura (Próximamente)

```
admin/
├── src/
│   ├── pages/         # Páginas del panel
│   ├── components/    # Componentes reutilizables
│   ├── services/      # Servicios (Firebase Admin, API)
│   ├── utils/         # Utilidades
│   └── styles/        # Estilos
└── public/            # Archivos estáticos
```

## 🎯 Funcionalidades

- 📊 Dashboard con estadísticas
- 👥 Gestión de usuarios
- 🐾 Gestión de mascotas
- 📝 Moderación de posts de comunidad
- 🛍️ Gestión de productos
- 📈 Analíticas y reportes
- 🔒 Control de acceso y permisos

## 🔧 Instalación

```bash
cd admin
npm install
```

## ▶️ Ejecutar

```bash
npm run dev
```

## 🔐 Acceso

Solo usuarios con rol de administrador pueden acceder a este panel.
