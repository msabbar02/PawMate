# 🌐 PawMate Web Application

Aplicación web para navegadores desarrollada con React/Next.js.

## 🚀 Tecnologías

- React / Next.js
- Firebase
- Tailwind CSS / Material-UI
- React Router

## 📂 Estructura (Próximamente)

```
web/
├── src/
│   ├── pages/         # Páginas de la aplicación
│   ├── components/    # Componentes reutilizables
│   ├── services/      # Servicios (Firebase, API)
│   ├── utils/         # Utilidades
│   └── styles/        # Estilos globales
└── public/            # Archivos estáticos
```

## 🎯 Funcionalidades

- Versión web de la app móvil
- Responsive design
- Acceso desde cualquier navegador
- Sincronización con app móvil

## 🔧 Instalación

```bash
cd web
npm install
```

## ▶️ Ejecutar

```bash
npm run dev
```
